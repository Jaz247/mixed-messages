# Mixed Messages
##### mixed-messages
##### Codecademy Portfolio Task

This program is designed to pass the Mixed Messages portfolio task on Codecademy.

## Project Outline

The task given as follows:

For this project, you will build a message generator program. Every time a user runs a program, they should get a new, randomized output. You’re welcome to take the project in a couple of different forms, like an astrology generator, inspirational message, or nonsensical jokes. To make your program truly random, the message that it outputs should be made up of at least three different pieces of data. Take what you know of JavaScript syntax so far to build the program and customize it to your liking.

To accomplish the requirements, we are going to use this scope:

+ Create an array that will store a list of verbs.
+ Create an array that will store a list of nouns.
+ Create an array that will store a list of adjectives.
+ Create an array that will store a list of sentence fragments that can be joined together (eg. "Today I went...", "You see a...").
+ Randomly select elements from each array and join these together to create an entirely new and randomised sentence.
